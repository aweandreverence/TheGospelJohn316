# The Gospel - John 3:16

A Gospel-sharing app built on React Native. Runs on web and mobile.

## Getting Started

### Prerequisites

1. Install `npm` (https://www.npmjs.com/)
1. Install `yarn`
    `[sudo] npm install -g yarn`

### Install the app

1. `make install`
